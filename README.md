#  Breast Cancer Data Analysis

##  Project Overview
This project analyzes a breast cancer dataset to explore patterns and build insights using Python.

##  Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

##  Key Steps
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Visualization of patterns
- Model building

##  Insights
- Certain features strongly correlate with diagnosis
- Visualization shows clear separation between malignant and benign cases

## Dataset
- Source: Kaggle – Breast Cancer Wisconsin (Diagnostic) Dataset
- File: `data.csv`
